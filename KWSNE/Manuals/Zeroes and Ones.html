On the Subject of Zeroes and Ones

This time, the revolution will be computerised.

This is the Internet. Say "Hi, Internet!"

The display will cycle through three different information types, which will tell you how you should navigate the Internet.

One of the Internet's nodes is the starting node and another is the goal node. Your task is to navigate from the former to the latter, moving exclusively between connected nodes. The Internet wraps around.

Once a suitable path has been found, press the display to reveal a keypad.

For each direction in the path, convert it into a pair of bits (where up = 00, right = 01, down = 10 and left = 11). Concatenate these bits and convert this binary number into decimal.

Enter this number, then press the submit button. The clear button will empty your input, unless your input is already empty, in which case the screen will revert to displaying information.

Submitting a valid number will solve the module. Submitting a number whose path does not end on the goal node, moves between unconnected nodes or is otherwise invalid will cause a strike. The submit button does nothing when your input is empty.

An option in this module's mod settings can be enabled, which will remove all animations that include motion. Where motion is needed as information, an arrow will show which way the motion is in.

Four Words (Starting Node)

[This information type will be on the module for certain.]

The displayed words are headings from Notepad++.

Modify a variable X according to the headers' modifications, taken in reading order on the module, in the table below. Assume that X has an initial value of the first digit of the serial number, plus ten.

Heading Modification
File Replace the first digit of X with a nine. If this did nothing, replace it with an eight instead.
Edit Reverse X. If this did nothing, add 1.
Search Add the number of modules on the bomb to X. Modules released before X-Ray count as three modules. For this modification, needies count as modules.
View Multiply X by the number of modifications that have been applied so far, plus one.
Encoding Convert the first letter of the serial number into a number (using A0Z25), then take this value modulo 10 and add two. Multiply X by this value.
Language Add the number of letters in the bomb's serial number to X. Vowels count as three letters, where the vowels are A, E, I, O and U.
Settings Add X's binary representation, reversed.
Tools Add the number of battery holders on the bomb, plus one, to X. D battery holders count as three battery holders.
Macro Take X's digital root and multiply it by eleven.
Run Repeat the previous modification. If this is the first heading, do the modification related to the next heading.
Plugins Multiply X by the number of ports on the bomb, plus one. If this did nothing, add this value to X instead.
Window Multiply X by the difference between the number of lit indicators on the bomb and the number of unlit indicators on the bomb, plus two.
? If the bomb knows your IP address, add the sum of the four numbers in your IP address, plus one, to X. Otherwise, if the voltage of the bomb is known, multiply X by the voltage of the bomb, rounding the voltage up. Otherwise, add 1 to X until it appears in the Fibonacci sequence.

Modulo X by the number of nodes in the Internet and add one. Then, from the Xth node in reading order, move to the Nth node in reading order that is connected to this node, where N is the last digit of the bomb's serial number, modulo the number of these connected nodes, plus one. The node you moved to is the starting node. For example, if X is 10 and N is 3, the starting node is π.

Binary (Dead Node)

The direction that the text box containing binary moves determines which row of the Internet the dead node is in (up = 1, right = 2, down = 3, left = 4).

The column that the dead node is in is determined by the binary.

  • There will be four strings of binary, each with a length of six bits.
  • Three of these strings are identical and have been offset by different amounts. The fourth, however, differs by one bit.
  • The position of this bit from left to right / top to bottom is the column of the dead node.

The path cannot pass through the dead node and may only enter the same row / column as the dead node once.
